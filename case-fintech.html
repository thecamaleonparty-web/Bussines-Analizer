The Architecture: Decoupled Intelligence

The application separates deterministic financial logic from probabilistic strategic analysis. This architectural decision enables:

  • Instant Response: Financial calculations execute in milliseconds without network latency.
  • Client Privacy: All sensitive financial data remains in the browser. Nothing hits a server.
  • AI-Ready Foundation: Modular architecture allows seamless integration with OpenAI API when needed.
