La Arquitectura: Inteligencia Desacoplada

La aplicación separa la lógica financiera determinista del análisis estratégico probabilístico. Esta decisión arquitectónica permite:

  • Respuesta Instantánea: Los cálculos financieros se ejecutan en milisegundos sin latencia de red.
  • Privacidad del Cliente: Todos los datos financieros sensibles permanecen en el navegador. Nada toca un servidor.
  • Base Lista para IA: La arquitectura modular permite integración perfecta con OpenAI API cuando sea necesario.
